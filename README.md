# pihome_display
Small pi based display for home data

##  Setup:  
Run this: https://www.waveshare.com/wiki/3.5inch_RPi_LCD_(B)  
with 99-fb-therbo config need to run:  
sudo apt-get update && sudo apt-get install xserver-xorg-video-fbturbo  

Make a folder called /home/pi/python  
setup venv to be at /home/pi/python/venv  
cd into /home/pi/python  and git clone this repo. 

# Run  
/home/pi/python/venv/bin/python /home/pi/python/pihome_display/main.py
